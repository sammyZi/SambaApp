# Nunito Font Assets

This directory contains the Nunito font family files required for the Samba File Browser App.

## Download Instructions

1. Visit the official Nunito repository: https://github.com/googlefonts/nunito
2. Download the following TTF files from the `fonts/TTF` directory:
   - Nunito-Regular.ttf
   - Nunito-Bold.ttf
   - Nunito-SemiBold.ttf
   - Nunito-Light.ttf

3. Place the downloaded TTF files in this directory (`assets/fonts/`)

## Alternative: Google Fonts

You can also download Nunito from Google Fonts:
1. Visit https://fonts.google.com/specimen/Nunito
2. Click "Download family"
3. Extract the TTF files from the downloaded ZIP
4. Copy the required weight files to this directory

## Required Files

- Nunito-Regular.ttf (400 weight)
- Nunito-SemiBold.ttf (600 weight)
- Nunito-Bold.ttf (700 weight)
- Nunito-Light.ttf (300 weight)
